Part 3B: Practical SVG

The idea

Hans Rosling did a very interesting TED talk about population growth with Ikea boxes and an animated graph. The animated graph was made with a Flash application from http://www.gapminder.org. Since data are freely available on the website, I just had the idea to build the same kind of animated graph with SVG, showing the variation in time of the two elements: GDP/capita and an happiness index for the 25 OECD countries.

Data extraction and transformation

I extracted the following data provided by GapMinder:
  • Total population from 1950 to 2050 (projection).
  • Income per person (GDP/capita, inflation-adjusted $)

I also retrieved the HPI index from http://www.happyplanetindex.org/public-data/files/hpi-2-0-results.xls. Given available data, I choosed to keep data between 1961 and 2005, with the ability to choose between the HPI or one of its components: Life Expectancy, Life Satisfaction, Ecological Footprint.

With regular expressions, I transformed original data into those 6 JSON files:
  • countries.json
  • ecologicalFootprint.json
  • gdpPerCapita.json
  • happyPlanetIndex.json
  • lifeExpectancy.json
  • lifeSatisfaction.json
  • population.json

The first JSON file is a one dimension array containing country labels. All the other JSON files have two dimensions: the year and the country. Since years are from 1961 to 2005, the actual year in the table is 1961 + <table index>. The second level tables contains the actual data for each country, in the same order as depicted in countries.json file.
